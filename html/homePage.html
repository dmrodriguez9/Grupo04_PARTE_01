Introducción Dom

Se podría decir que las etiquetas HTML son la columna vertebral de un documento.

Según el Document Object Model (DOM), cada etiqueta HTML es un objeto, incluso el texto dentro de una etiqueta es un objeto. Todos estos objetos son accesibles usando JavaScript, y podemos utilizarlos para modificar nuestra página.

Métodos

Los métodos HTML DOM son acciones que puede realizar (en elementos HTML). Las propiedades HTML DOM son valores (de elementos HTML) que puede establecer o cambiar.

Componentes

El objeto del documento HTML DOM es el propietario de todos los demás objetos en su página web, distinguiendolo de otros sistemas que aparentemente pueden resultar similares a él.

Elementos

Los elementos que puedan estar dentro de la página, como párrafos, divisiones, tablas, formularios y sus campos, etc. A través del DOM se puede acceder, por medio de Javascript, a cualquiera de estos elementos, es decir a sus correspondientes objetos para alterar sus propiedades o invocar a sus métodos.

Html

El HTML DOM permite que JavaScript cambie el contenido de los elementos HTML. La forma más sencilla de modificar el contenido de un elemento HTML es mediante la innerHTML propiedad.

CSS

El HTML DOM permite que JavaScript cambie el estilo de los elementos HTML en cada una de las páginas web que estemos creando, con el fin de modificar, tanto su estructura, estilo y contenido.
